对于使用 Mac OS X 的用户
在任务栏点击 Docker for mac 应用图标-> Perferences…-> Daemon-> Registrymirrors。在列表中填写加速器地址 https://registry.docker-cn.com 。
修改完成之后，点击 Apply&Restart 按钮，Docker 就会重启并应用配置的镜像地址了。
